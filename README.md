# Energy Simulator

A Next.js application for simulating energy consumption and storage in buildings.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Simulate energy flow over time
- Visualize battery state of charge (SOC)
- Manage buildings and tenants

## Deployment

This app can be deployed on Vercel, Netlify, or Render for free.
